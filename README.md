# Mon Profil Personnel

## Description

Ce projet a été réalisé dans le cadre d'un exercice d'apprentissage du HTML5 sémantique et du CSS3.

L'objectif est de créer une page web personnelle présentant :

- Une structure HTML sémantique
- Une navigation principale
- Une section de présentation
- Une liste de compétences
- Des informations complémentaires
- Un formulaire de contact
- Une mise en page moderne avec CSS

## Technologies utilisées

- HTML5
- CSS3
- Visual Studio Code
- Git & GitHub
